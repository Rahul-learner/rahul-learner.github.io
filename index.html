Native Drag and Drop
Dragging files directly onto the website

Drag external images into the container below

You are using Internet Explorer. IE only supports the 'text' and 'URL' transfer types so your experience may be limited. Also IE9 and below have no support for dragging images from your desktop to the page

Any draggable elements (images, link tags, selected text) can be dropped here, but only dragged images from external sites / uploaded images will be displayed

Some images are wrapped around an anchor tag, these won't work as the API will consider the drop to be a 'link' and not an 'image'

Notes

  • Most images online will be wrapped around a <a> tag. When you drag and drop these it looks like you're dragging the image but actually you're dragging the link (which won't work below). This is a limitation of the API
  • There is no concept of what an "image" is to the API. As such we need to check to see if the URI supplied has a filetype in it. We check the URL to see if it contains the '.jpg', '.jpeg', '.png' ,'.bmp', or '.gif' extension
  • Some CND's use strange URL patterns that don't end in a specific file type. These won't work as the API won't be able to tell what it is.
